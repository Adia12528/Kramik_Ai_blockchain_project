## üõ†Ô∏è What Services You'll Use

### 1. MongoDB Atlas (Database)
- **What**: Cloud MongoDB database
- **Why**: Free 512MB, no credit card needed
- **Signup**: [mongodb.com/cloud/atlas](https://www.mongodb.com/cloud/atlas/register)
- **Time**: 5 minutes

### 2. Render.com (Backend)
- **What**: Node.js hosting platform
- **Why**: Free 750h/month, better than serverless
- **Signup**: [render.com](https://render.com)
- **Time**: 10 minutes

### 3. Vercel (Frontend)
- **What**: React/Vite deployment platform
- **Why**: Free, fast, automatic HTTPS
- **Signup**: [vercel.com](https://vercel.com)
- **Time**: 5 minutes

### 4. Cloudinary (File Storage)
- **What**: Cloud file storage & CDN
- **Why**: Free 25GB, easy integration
- **Signup**: [cloudinary.com/users/register_free](https://cloudinary.com/users/register_free)
- **Time**: 5 minutes

### 5. Google AI (Gemini API)
- **What**: AI/ML API for chatbot
- **Why**: Free 1,500 requests/day
- **Get Key**: [ai.google.dev](https://ai.google.dev/gemini-api/docs/api-key)
- **Time**: 2 minutes

### 6. Sepolia Testnet (Blockchain)
- **What**: Ethereum test network
- **Why**: Free test ETH, no real money
- **Faucet**: [sepoliafaucet.com](https://sepoliafaucet.com)
- **Time**: 5 minutes

## üÜò Need Help?

### Quick Fixes

**Backend not responding?**
- Check MongoDB connection string
- Verify environment variables in Render
- Check Render logs

**Frontend can't connect to backend?**
- Update VITE_API_URL in Vercel
- Check CORS settings in backend
- Redeploy frontend

**Blockchain not working?**
- Check you're on Sepolia network
- Verify you have test ETH
- Check contract addresses

### Full Troubleshooting

Each guide has a detailed troubleshooting section:
- Common errors and solutions
- How to check logs
